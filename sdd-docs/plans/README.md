## 문서 작성 가이드

각 계획서에는 다음 내용을 포함합니다:

1. **개요** - 기능 목적과 배경
2. **사용자 플로우** - UX 흐름도
3. **컴포넌트 구조** - 필요한 컴포넌트와 Props/State
4. **파일 구조** - 생성/수정할 파일 목록
5. **구현 순서** - 단계별 작업 계획
6. **체크리스트** - 완료 여부 추적
