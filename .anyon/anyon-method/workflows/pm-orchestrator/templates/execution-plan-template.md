## 6. 실행 가이드

### Wave 실행 규칙

1. **순차 실행**: Wave 1 → Wave 2 → Wave 3 → ... 순서로 진행
2. **병렬 실행**: 같은 Wave 내 `parallel_with` 티켓들은 동시 실행 가능
3. **완료 확인**: Wave 내 모든 티켓 완료 후 다음 Wave 진행
4. **의존성 체크**: 각 티켓의 `blocked_by` 티켓들이 모두 완료되었는지 확인

### 티켓 실행 방법

각 티켓은 execution-plan.md의 Wave 섹션에 정의되어 있습니다:

1. **담당 에이전트**: 순서대로 작업 수행
2. **참조 문서**: 작업에 필요한 설계 문서 섹션
3. **수용 기준**: 완료 확인 체크리스트
4. **산출물**: 생성해야 할 파일 목록
