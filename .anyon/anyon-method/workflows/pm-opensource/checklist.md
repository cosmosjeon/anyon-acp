# PM Opensource Checklist Validation

## 문서 로드

- [ ] open-source.md 파일이 존재함
- [ ] 문서에서 최소 1개 이상의 GitHub URL 추출됨
- [ ] 각 레포지토리의 용도/설명이 파악됨

## Clone 실행

- [ ] 현재 작업 디렉토리가 올바르게 식별됨
- [ ] 각 레포지토리에 대해 존재 여부 확인 수행됨
- [ ] 기존 존재하는 폴더는 skip 처리됨
- [ ] git clone 명령이 올바르게 실행됨
- [ ] 실패한 clone에 대해 에러 메시지 기록됨

## 문서 업데이트

- [ ] open-source.md 문서에 clone 경로가 추가됨
- [ ] 각 레포지토리의 상태(완료/skip/실패)가 기록됨
- [ ] 문서가 올바르게 저장됨

## 결과 보고

- [ ] 전체 clone 결과 요약이 표시됨
- [ ] 성공/skip/실패 통계가 정확함
- [ ] 다음 단계 안내가 포함됨
