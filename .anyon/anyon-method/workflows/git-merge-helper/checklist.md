# Git Merge Helper 검증 체크리스트

## 사전 검증

- [ ] 현재 디렉토리가 Git 저장소인지 확인됨
- [ ] 소스 브랜치가 존재함
- [ ] 타겟 브랜치가 존재함
- [ ] 타겟 브랜치로 체크아웃 완료

## 머지 실행

- [ ] `git merge --no-commit` 실행됨
- [ ] 충돌 여부 확인됨
- [ ] 충돌 파일 목록 수집됨 (충돌 발생 시)

## 충돌 해결

- [ ] 모든 충돌 파일이 처리됨
- [ ] 각 충돌에 대해 사용자 선택을 받음
- [ ] 충돌 마커(<<<<<<, =======, >>>>>>)가 모두 제거됨
- [ ] 선택에 따른 코드 수정이 올바르게 적용됨

## 머지 후 검증

- [ ] 존재하지 않는 함수/변수 호출 검사 완료
- [ ] 누락된 import 문 검사 완료
- [ ] 중복 코드 검사 완료
- [ ] 발견된 문제에 대해 수정 제안 제시됨
- [ ] 사용자 승인 후 수정 적용됨

## 완료

- [ ] 최종 변경사항 요약 표시됨
- [ ] 커밋 메시지 생성/입력됨
- [ ] `git add .` 및 `git commit` 실행됨
- [ ] 완료 메시지 표시됨

## 사용자 경험

- [ ] 모든 설명이 초보자도 이해할 수 있게 작성됨
- [ ] 전문 용어에 쉬운 설명이 덧붙여짐
- [ ] 각 선택지의 결과가 명확히 설명됨
- [ ] 비유와 이모지를 적절히 사용하여 친근하게 전달됨
